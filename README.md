# Lyz's Blog
这是hexo博客框架所存储的博客内容
Website: https://blog.lyz05.cn/

# 快速开始
## 依赖安装
```bash
git clone git@github.com:lyz05/hexo
cd hexo
npm i hexo-cli yarn -g
npm install
```

## 运行项目
```bash
npm run build   # 普通构建
npm run clean   # 清理配置信息
npm run server  # 本地调试
npm run douban  # 生成豆瓣电影图书信息
```

# 检查组件更新
升级package.json
```bash
yarn upgrade-interactive --latest
npm outdated
npm update
```
